6. Express 웹서버 만들기
6.1 Express 프로젝트 만들기

1) express generator 사용
2) 직접 구현하기

* express 프로젝트 시작하기
1) npm init ==> package.json
2) express 패키지 설치
    npm i express
3) nodemon 명령어 설치
    node.exe monitor라고 생각하면 됨. 수정시 서버의 재시작 자동화
    npm -i -D nodemon
4) app.js 작성
5) npm start 서버 실행
6) http://localhost:3000/ 접속 테스트

7) 폴더 작성
views, models, routes
